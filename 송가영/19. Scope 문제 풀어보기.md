### 문제 1. 다음 코드가 console에 출력할 내용은?  

```javascript
for(var i =0; i<5; i++) {
  setTimeout(function() {
    console.log(i);
  }, 1000);
}
```

<br/>

### 문제 2. 다음 코드가 console에 출력할 내용은?  

```javascript
for(let i =0; i<5; i++) {
  setTimeout(function() {
    console.log(i);
  }, 1000);
}
```

<br/>

### 문제 3. 다음 코드가 console에 출력할 내용은?  

```javascript
for(let i =0; i<5;) {
  setTimeout(function() {
    console.log(i);
  }, 1000);
  
  i++;
}
```

<br/>
